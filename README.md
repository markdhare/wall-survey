# Wall Survey

Wall Survey is a PySide6 desktop application for organizing and analyzing NanoVNA S21 measurements on a physical wall grid. It imports Touchstone files, compares measurements with reference runs, and presents RF metrics as tables, traces, and spatial heatmaps.

## Quick start

```powershell
uv sync --extra dev
uv run wall-survey
```

See [documentation/USER_GUIDE.md](documentation/USER_GUIDE.md) for the complete workflow and [documentation/DEVELOPER_GUIDE.md](documentation/DEVELOPER_GUIDE.md) for architecture and algorithms.
