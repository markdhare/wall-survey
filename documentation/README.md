# Wall Survey Documentation

Start with the [User Guide](USER_GUIDE.md). It explains the front-view coordinate convention, recommended scientific workflow, Run Lab/reference/map distinctions, analysis choices, and a beginning-to-end first survey tutorial.

Use the [Direct Acquisition Guide](ACQUISITION_GUIDE.md) while connecting a NanoVNA-H or NanoVNA-H4. It covers serial-port ownership, firmware constraints, sweep segmentation, averaging, immediate raw preservation, routing, guided grid progression, quality warnings, and recovery behavior.

The [File Formats Guide](FILE_FORMATS.md) documents Touchstone support, portable `.wallscan` archives, `project.yaml`, direct-acquisition `capture_log.yaml`, and CSV exports. The [Developer Guide](DEVELOPER_GUIDE.md) describes architecture, RF calculations, hardware-driver boundaries, tests, and planned extension points. [Release Notes](RELEASE_NOTES.md) identify stable milestones and compatibility changes.

The separate [project article outline](../article/wall_mapping_project_outline.md) is a journal-inspired narrative skeleton for documenting the experiment, its limitations, and possible applications.
